A co dál?

Určitě nezapomeň navšívit oficiální web SOČky http://www.soc.cz/. Za zmínku stojí také web http://www.sockari.cz/, který podává všechny důležité informace pro každého SOČkaře na jednom místě.

Spousta (nejen) našich SOČek je podpořená organizací JCMM. Určitě koukni na jejich web http://www.jcmm.cz/projekt/soc_studenti, můžeš si tam prohlédnout konkrétní podpořená téma a pokud stále váháš, nějaké si vybrat.
